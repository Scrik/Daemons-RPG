  Daemons RPG

Описание:
 Плагин для Minecraft сервера +1.8 версии. (доработаю)
 
Как установить плагин?
  1. Скачиваем последнию сборку с папки "build".
  2. Сборку плагина , кладем по данному пути "youServer/plugins/".
  3. Запускаем сервер.
